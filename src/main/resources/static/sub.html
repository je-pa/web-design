포티넷은 통신업체, 데이터 센터, 기업 및 지사용 네트워크 보안 어플라이언스를 공급하는 글로벌 선도 기업입니다.

제품소개

"포티넷의 사명은 IT인프라를 보호하고 간소화하여 가장 혁신적이며 최고 성능을 자랑하는 네트워크 보안 플랫폼을 제공하는 것입니다."

특장점

  • FAST

    비즈니스 과제
    - 기업 대역폭 요구 사항은 18개월 단위로 2배로 증가
    - 기존 보안 솔루션이 걸림돌
    - 중요한 비즈니스트래픽속도의 저하, 사용자 불만 및 신규 프로젝트 지연
    포티넷의 장점
    - 고성능 맞춤형 FortiASIC 프로세서를 통한 성능 강화로 경쟁력이 5배 이상 강화된 솔루션 제공
    - 성장하는 대역폭 요구에 대응할 수 있는 보안 제공
    - 비즈니스 성장 속도 가속화 및 신규 이니셔티브 개시
    - 사용자 경험 향상

  • SECURE

    비즈니스 과제
    - 더욱 더 정교해지고 위험성이 커진 오늘날의 위협으로 보안 됨에 대한 경영진의 압박 증가
    - 대부분의 보안 첩체가 한층 강화된 위협에 대응하기 위한 필수 요소를 모두 공급하지 못하므로 여러 솔루션의 조합이 요구됨
    포티넷의 장점
    - 대규모의 인하우스 FortiGuardLabs 글로벌 위협 리서치 팀 운영
    - 풍부한 독점통합 보안 기술 보유
    - 연중무휴(24x365)로 제공되는 보안 업데이트
    - 오늘날의 위협에 대한 우수한 효과를 독립적으로 입증

주요기능

  • FortiGate

    Fortigate 차세대 방화벽은 단일 솔루션을 적용함으로써 보안환경을 간소화합니다.

  • fortiSandbox

    FortiSandbox는 지능형 위협 공격으로부터 신속한 보호를 제공합니다.

  • FortiAnalyzer

    FortiAnalyzer는 네트워크 위협, 네트워크 보안 로깅 등을 신속하게 분석할 수 있습니다.

  • FortiManager

    FortiMabager는 보안 인프라 중앙 집중식 관리와 제어를 할 수 있습니다.

  • FortiWeb

    FortiWeb은 안성코드, OWASP Top10등에서 웹애플리케이션을 보호합니다.

  • FortiSIEM

    FortiSIEM은 단일 애플리케이션에서 통합 보안, 성능 및 가용성을 모니터링합니다.
